# Template for E2E typescript monorepos

Running at [https://frontend-z0en.onrender.com/](https://frontend-z0en.onrender.com/)

Build end to end typescript apps with:
- PNPM workspace
- Turborepo
- Next.js
- Vite
- eslint
- zapper
- zod
- superjson
- Vitest
- tRPC
- shadcn/ui
- Clerk (extension)
- BullMQ (extension)
